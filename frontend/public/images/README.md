# Dossier Images

Placez votre logo de la plateforme Rapido dans ce dossier avec le nom `logo.png`.

Le logo sera automatiquement affiché sur la page de bienvenue.

**Format recommandé :**
- Nom du fichier : `logo.png`
- Taille recommandée : 300x300px minimum
- Format : PNG avec transparence (optionnel)

Une fois le logo ajouté, il apparaîtra automatiquement sur la première étape de la page de bienvenue.

**App Android (Capacitor)** : les icônes et le splash natifs sont générés à partir de ce même `logo.png` (voir `scripts/android-brand-assets.sh` après un `npx cap sync` si les mipmaps sont réécrites).
