# Dossier Illustrations

Placez vos images d'illustrations pour la page de bienvenue dans ce dossier.

## Fichiers requis :

1. **welcome.png** - Illustration pour la première étape "Bienvenue sur Rapido"
2. **delivery.png** - Illustration pour la deuxième étape "Livraison rapide"
3. **start.png** - Illustration pour la troisième étape "Commencez maintenant"

## Format recommandé :

- **Format** : PNG avec transparence (recommandé) ou JPG
- **Taille** : 600x600px minimum
- **Style** : Illustrations colorées et modernes qui s'intègrent au design
- **Poids** : Optimiser les images (max 500KB par image)

## Utilisation :

Les illustrations seront automatiquement affichées avec des animations interactives et des mini stickers animés autour d'elles.

Une fois les images ajoutées, elles apparaîtront automatiquement sur la page de bienvenue avec de superbes animations !
