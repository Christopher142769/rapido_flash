# Dossier Bannières

Placez vos images de bannières pour le carrousel mobile dans ce dossier.

## Fichiers requis :

1. **banner1.jpg** - Première bannière
2. **banner2.jpg** - Deuxième bannière
3. **banner3.jpg** - Troisième bannière
4. **banner4.jpg** - Quatrième bannière

## Format recommandé :

- **Format** : JPG ou PNG
- **Taille** : 1080x400px (ratio 2.7:1)
- **Style** : Images de plats appétissants, promotions, ou offres spéciales
- **Poids** : Optimiser les images (max 300KB par image)

## Utilisation :

Les bannières défilent automatiquement toutes les 4 secondes sur la page d'accueil mobile. Les utilisateurs peuvent également cliquer sur les indicateurs en bas pour naviguer manuellement.

Une fois les images ajoutées, elles apparaîtront automatiquement dans le carrousel !
